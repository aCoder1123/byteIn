What's New?

Your table management system now includes Firebase integration for storing and retrieving custom maps!

  • Save custom maps to Firebase cloud storage
  • Load any previously saved map from anywhere
  • Share maps across different devices and users
  • Persistent storage that survives browser refreshes
  • Map selection dropdown with floor information

How It Works

When you create a custom map with the map generator:

  1. Create: Upload image, add table components, customize layout
  2. Save to Firebase: Click "Save to Firebase" and give your map a name
  3. Access Anywhere: Anyone can now select and load this map from the dropdown
  4. Manage Tables: Use the loaded map in "Manage Tables" view with full interactivity

Setup Instructions

To enable Firebase functionality:

  1. Create a Firebase project at Firebase Console
  2. Enable Firestore Database
  3. Update firebase-config.js with your project configuration
  4. Set Firestore rules for development (see FIREBASE_SETUP.md)
Note: The app works without Firebase too! Local storage and file download/upload are still available for offline use.

Data Structure

Maps are stored in Firebase with this structure:

{ name: "Restaurant Floor 1", floor: 1, image: "data:image/jpeg;base64...", // Base64 encoded image components: [ { id: "table-1", type: "table", name: "Table 1", x: 100, y: 150, occupied: false } ], createdAt: "2024-01-15T10:30:00.000Z", updatedAt: "2024-01-15T10:30:00.000Z" }
